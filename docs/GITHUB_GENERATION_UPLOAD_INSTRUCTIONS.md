# GitHub upload instructions for generation formulation

1. Open your GitHub repository: `vMF-hypersphere-music`
2. Open the `docs/` folder.
3. Upload `generation_formulation.md`.
4. Commit message: `Add generation formulation documentation`
5. Open `README.md` and add the contents of `README_generation_snippet.md`.
